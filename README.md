# EZ Management
Inventory system with order function and receipt printing. 

Built with Springboot 3 and Java 21.

## Required
1. Install MySQL 8.1 Database Server or any database server you prefer
2. Install IntelliJ IDEA or any IDE you prefer

## Set-up
1. Configure `src>main>resources>application.properties`
   * Create database
   * Update database address and credentials
2. Run `mvn install`
3. Run `mvn spring-boot:run`
